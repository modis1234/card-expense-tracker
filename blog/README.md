## 포스팅 업로드 방법

### Velog에 업로드하기

1. [Velog](https://velog.io) 로그인
2. 우측 상단 "새 글 작성" 클릭
3. 마크다운 파일 내용 복사 & 붙여넣기
4. 시리즈 설정: "카드 가계부 만들기"
5. 태그 추가: `프로젝트`, `NestJS`, `Next.js`, `AI`, `가계부`
6. 발행

### Tistory에 업로드하기

1. Tistory 관리자 페이지 접속
2. "글쓰기" 클릭
3. 마크다운 모드로 전환
4. 마크다운 파일 내용 복사 & 붙여넣기
5. 카테고리: "프로젝트" 또는 "개발"
6. 태그 추가
7. 발행

## 포스팅 작성 원칙

### 구성
- **들어가며**: 문제 정의, 동기
- **본문**: 해결 방법, 코드, 설명
- **마무리**: 다음 단계 예고

### 스타일
- 코드는 최소한으로 (핵심만)
- 이미지/다이어그램 활용 (선택적)
- 실제 겪은 문제와 해결 과정 중심
- 기술적 깊이보다는 실용성 중심

### 시리즈 연결
- 각 글 끝에 "다음 포스팅 예고" 추가
- 시리즈 목록 링크 추가
