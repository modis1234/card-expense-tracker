# 카드사 샘플 수집 가이드

## 샘플 파일 수집 방법

### 1. 본인 카드 이용내역 다운로드
각 카드사 웹사이트/앱에서 엑셀 파일 다운로드

### 2. 샘플 파일 저장 구조
```
samples/
├── shinhan/
│   ├── sample1.xlsx
│   └── sample2.xlsx
├── samsung/
│   ├── sample1.xlsx
│   └── sample2.xlsx
├── hyundai/
│   ├── sample1.xlsx
│   └── sample2.xlsx
├── kb/
│   ├── sample1.xlsx
│   └── sample2.xlsx
└── lotte/
    ├── sample1.xlsx
    └── sample2.xlsx
```

### 3. 개인정보 처리
**중요: 샘플 파일에서 개인정보 제거 필수**
- 카드번호 마스킹
- 실명 제거
- 민감한 가맹점명 수정 (병원명 등)

## 카드사별 다운로드 경로

### 신한카드
- 웹사이트: https://www.shinhancard.com
- 경로: 로그인 → 이용내역 → 엑셀 다운로드
- 앱: 신한 쏠(SOL) → 카드 → 이용내역

### 삼성카드
- 웹사이트: https://www.samsungcard.com
- 경로: 로그인 → 이용내역 조회 → 엑셀 저장
- 앱: 삼성카드 앱 → 이용내역

### 현대카드
- 웹사이트: https://www.hyundaicard.com
- 경로: 로그인 → 이용내역 → 다운로드
- 앱: 현대카드 앱 → 이용내역

### KB국민카드
- 웹사이트: https://www.kbcard.com
- 경로: 로그인 → 이용내역 → 엑셀 다운로드
- 앱: KB Pay 앱 → 카드 → 이용내역

### 롯데카드
- 웹사이트: https://www.lottecard.co.kr
- 경로: 로그인 → 이용내역 → 엑셀 저장
- 앱: 롯데카드 앱 → 이용내역

## 분석 체크리스트

각 샘플 파일에 대해 다음 항목 확인:

- [ ] 파일 확장자 (.xlsx, .xls, .csv)
- [ ] 시트 개수 (단일/복수)
- [ ] 헤더 위치 (몇 번째 행)
- [ ] 데이터 시작 위치
- [ ] 컬럼 개수
- [ ] 필수 컬럼 존재 여부 (날짜, 가맹점, 금액)
- [ ] 날짜 포맷
- [ ] 금액 포맷 (쉼표, 통화기호)
- [ ] 병합된 셀 여부
- [ ] 요약 행/푸터 존재 여부
- [ ] 빈 행 존재 여부
- [ ] 특수 문자 사용 여부

## 참고사항

### 샘플이 없는 경우
- 웹 검색으로 카드사 엑셀 포맷 정보 찾기
- 카드사 고객센터 문의
- 커뮤니티에서 샘플 요청 (개인정보 제거 조건)

### 테스트 데이터 생성
실제 샘플 확보가 어려운 경우, 가상의 테스트 데이터 생성:
```
날짜, 가맹점명, 금액, 할부
2024-01-15, 스타벅스 강남점, 5500, 일시불
2024-01-16, GS25 편의점, 3200, 일시불
2024-01-17, 쿠팡, 45000, 3개월
```
