# 와이어프레임 (Wireframes)

## 개요

이 문서는 카드 결제 내역 자동 분류 가계부 서비스의 주요 화면 와이어프레임을 포함합니다.

**디자인 원칙:**
- 단순하고 직관적인 인터페이스
- 명확한 정보 계층 구조
- 일관된 디자인 시스템
- 반응형 디자인 (모바일/태블릿/데스크톱)

**색상 팔레트:**
- Primary: #4F46E5 (Indigo)
- Secondary: #10B981 (Green)
- Accent: #F59E0B (Amber)
- Background: #F9FAFB (Gray 50)
- Text: #111827 (Gray 900)
- Border: #E5E7EB (Gray 200)
